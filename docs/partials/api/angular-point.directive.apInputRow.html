apInputRow
directive in module angular-point

Description

Creates individual form controls for each of the field names provided.

Usage

as attribute
<ANY ap-input-row
     entity="{object}"
     fields="{string[]|string}"
     container-class="{string}"
     ng-disabled="{boolean}">
   ...
</ANY>

Parameters

ParamTypeDetails
entityobject

SharePoint list item.

fieldsstring[]string

Names of the properties on the entity to create individual form fields for.

containerClassstring

Optional override for the class on the containing element.

(default: 'row')

ngDisabledboolean

Pass through to disable control using ng-disabled on element if set.

(default: false)
