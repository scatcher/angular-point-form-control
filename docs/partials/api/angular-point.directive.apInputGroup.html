apInputGroup
directive in module angular-point

Description

Creates individual form controls for each of the field names provided.

Usage

as attribute
<ANY ap-input-group
     cols="{function|number}"
     description="{string}"
     field-definition="{object}"
     [field-definition.-choices]="{string[]}"
     [field-definition.label]="{string}"
     [field-definition.object-type]="{string}"
     field-name="{string}"
     entity="{object}"
     [label]="{string}"
     ng-disabled="{boolean}"
     [validation]="{function}">
   ...
</ANY>

Parameters

ParamTypeDetails
colsfunctionnumber

Column width in a 12 column layout.

(default: 3)

descriptionstring

Optional description text.

(default: '')

fieldDefinitionobject

Optionally override the field definition stored in the model with a custom field definition.

(default: {Definition from model})

[fieldDefinition.Choices]string[]

Choices to appear in dropdown.

[fieldDefinition.label]string

Label for the input.

[fieldDefinition.objectType]string

One of the valid SharePoint field types.

fieldNamestring

The name of the property on the entity to bind to.

entityobject

SharePoint list item.

[label]string

Label for the input.

ngDisabledboolean

Pass through to disable control using ng-disabled on element if set.

(default: false)

[validation]function

Allow you to pass in validation logic.
